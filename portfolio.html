Portfolio

App: Eat A Burger!

An app deployed on Heroku, with html served by Handlebars, with data from MySQL. User can create a burger, and add it to a list and then devour it, which will move it to the devoured list.

Note App

A simple note taking app, where the user can save notes to time blocks, and the time blocks are colored based on current time. The user can also clear all the notes for the day easy re-use.

Weather Dashboard

A weather dashboard that allows a user to search a city for the current weather, and 5 day weather forecast. The last several cities are saved to a list that can be clicked on for quick revisiting. This list also saves to local storage.

RoamForth

A site that allows a user to select a state, then lists the National Parks available in that state. Once a park is selected, the user can see park info, pictures taken from the park, available activites, links to the park's web page, as well as directions, and a map that display the location of the park.

Forage

A web application that allows users to find out their daily calorie requirements based on goals and activity and then search foods that display ingredients and calories. The user can save their favorite items to their own personal database to be displayed any time.

Password Generator

A simple password generator that allows users to create a random password by selecting from four character types, and password length. There is also a copy password function.
